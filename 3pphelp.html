How do I use this?
To get started with this tool, find a spot where you want measure the strike, dip, or dip azimuth of the land and Alt+Click three times along the area. Using areas of different elevation that are in a relatively straight line will yield the best results. Alternatively, you can enter three points and elevations into the manual plot boxes and click Manual Plot.

For help with using the other features of this application check the other help topics.

What is the purpose of this tool?
The purpose of this tool is to allow interactive 3D three-point problem solving. It can be used to calculate planes, strike, dip, and many other geological features. A powerful part of this tool is that once the symbols have been placed they can be exported to Google Earth Desktop to be further manipulated and examined.

Who built this tool?
This tool was built by Jack Jamieson (Software Developer) and Greg Herman (Structural Geologist), with help from several sources on the web. These can be found in the references.

Fly to
Fly to allows you to go anywhere on the globe simply by typing in the name of that place. It works similarly to Google Maps and is fairly good at guessing what you are looking for. It also accepts latitude, longitude values.

URL Import
From the URL import box you can import a KMZ file that is hosted elsewhere on the internet. This is the best way to get imagery into this tool. This function was not built with importing KML in mind. There is a seperate local upload for KML.

Toggles
The toggles will turn on and off the historical imagery and borders/places. The globe pre-loads the borders and places but historical imagery is off by default.

Import KML
This button allows you to import a KML file from your local hard drive. If there are any images in the KML from your hard drive however, they will not work. Images MUST BE HOSTED ONLINE to be seen when using this tool.

Export Session
The export session button allows you to download your work from this tool into a KML file for easy import into Google Earth desktop.

Clear All Data
Clicking this button will clear EVERYTHING you have on the globe, including imported KML or KMZ content.

Manual Plot
Clicking manual plots takes the points from the boxes marked P1, P2, and P3 from the boxes below the buttons and plots those points on the globe. The Google Earth altitude values are not used and the calculations will be done based on your input for altitude.

Symbol Scale
Only 3D circles may be scaled. All other symbols are 2D and are auto-scaled based on the distance of your points. If no scale is chosen then the tool will approximate a scale for you. Otherwise, you can input numbers into x and y to scale the symbols in meters. Sometimes the elevation of the earth will obscure the 2D symbol. This can be fixed by exporting to KML and raising the symbol in GE Desktop.

Outputs
At the bottom of the sidebar are the outputs. These include the elevation, latitude, and longitude of your mouse position and the calculated strike, dip, dip azimuth, and quadrant after three points have been placed.
