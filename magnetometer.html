Magnetometer Tools

These tools are can be used in conjunction with the Magnetometer app for post processing of data.

Magnetometer Android App
Uses the magnetometer in your Android phone to record magnetic fields along with GPS and time series information. Not yet available.

Excel to KML for Magnetometer Data
The app can create its own KML, but if you only have a CSV file this will do it too.

Excel to Heatmap
Generates a heatmap based on the CSV data collected from the app.
