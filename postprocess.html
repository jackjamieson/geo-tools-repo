KML Placemark Digitizer

Help Docs

This tool is a GUI application that reads local or web based KML and KMZ files. It is best used with imagery baked in such as topographic maps so that the opacity can be changed. Placemarks can be generated quickly and with several options including sequential formats.

To begin using this tool you must download the ZIP file containing the executable. Extract the ZIP. Double click the .jar to open it and begin using it.
