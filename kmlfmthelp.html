USGS KML Formatter - Help

The KML Formatter will take USGS state KMLs from http://mrdata.usgs.gov/geology/state/ and put the layers into folders based on their Age, Lithic Group, and Age with Lithic Subgroups.

Instructions:

  1. Download the desired state KML from the link above. The website can also be found by clicking Help > USGS State KMLs in the program.
  2. Click File > Open File and select the state KML. An progress indicator will appear to the right of the Help tab but it should be fairly quick to process the file.



    3. Example of the folder structure and Feature description viewer.


  3. Preview the output in the left pane of the window. This is how the resulting KML will look depending on the Export options you select.
  4. Click File > Export as...(KML or KMZ) and choose the folder structure that you want.
