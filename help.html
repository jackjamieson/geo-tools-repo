Atlas Sheet/DDMMSS to Decimal Degree Converters - Help

The Atlas Sheet Conversion utilities help you convert between Atlas Sheet Coordinates (e.g. 22-05-1-2-3) and Decimal Degree and Degree Minute Second latitude and longitude in NAD 27. The batch version only converts Atlas Sheets into latitude longitudes. It's output can be formatted for use with NADCON to convert NAD 27 to NAD 83.

The help topics also apply to the Decimal Degree to DDMMSS converter.

What are the accepted inputs?

Only positive numbers are accepted. For latitude and longitude it is assumed they are N and W respectively.

For Atlas Sheet to Lat/Long:
There are three text boxes representing the different parts of the Atlas Sheet. It is looks like Sheet:Block:[Rectangle][Square][Unit]. For example, 22:12:111 would be 22-12-1-1-1 on an Atlas Sheet. All fields must be filled in completely, so if a block is 4 for example, you must enter 04 for the field to be complete.

For Lat/Long to Atlas Sheet:
You can use either the DDMMSS input or the Decimal Degree input. Clicking Calculate on either will output the result below it. DDMMSS only accepts 6 digits, no decimals. Decimal degree can have a variable amount of decimal places, but will output accurate to 7 decimals.

For Batch Atlas Sheet:
You must enter the Atlas Sheet in the format "##:##:###", similar to the single sheet conversion. Each new coordinate must be entered on a new line, one per line.

How do I use the batch tool?

Following the input rules found in "What are the accepted inputs?", the batch tool is used the same way as the single point calculations. When you are satisfied with what you have entered, hit Calculate and the results will be displayed in the Output box to the right.

If the checkbox "Format for NADCON" is not checked, the output will be formatted in a more readable format.

What is the "Format for NADCON" option?

If you would like to convert the Atlas Sheet batch output from NAD 27 to NAD 83, the tool can format the output for use with the NADCON converter tools found at http://www.ngs.noaa.gov/cgi-bin/nadcon.prl

The checkbox must be checked in order for the tool to output it in the correct format. When you are ready to bring the output into NADCON, click Download Output, go to NADCON, and make sure these settings are selected:
  • NAD 27 to NAD 83
  • Free Format 2
You may choose whichever output you want from NADCON.

This checkbox must also be checked in order to export the file as KML!
